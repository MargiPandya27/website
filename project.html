    1. Multiple Foreground Co-segmentation
      Advisor: Dr. Avik Hati
      Multiple foreground common object co-segmentation is a task in computer vision with applications ranging from image understanding to object recognition. The goal is to segment common objects shared among multiple images within a given group. Till now, research in this area has primarily focused on single foreground co-segmentation, where only a single common object is segmented across multiple images. In this project, we work on key approaches such as deep learning-based methods, graph-based algorithms, and optimization techniques to develop an algorithm advised for multiple foreground co-segmentation.     2. Contrastive Fusion of Transformer and CNN based Deep Features for Semantic Segmentation
      Advisor: Dr. Tushar Sandhan
      Image segmentation is crucial in various computer vision applications for accurately identifying and extracting objects or regions of interest within an image. Despite significant advancements in state-of-the-art (SOTA) models for semantic segmentation, many still rely on additional datasets for improved performance. We introduce ContraFusionNet, a novel segmentation architecture combining the spatial understanding property of Convolutional Neural Network (CNN) and deep feature extraction capability of attention in transformers for robust semantic segmentation even in limited data. It uses student-teacher learning with contrastive loss for improved feature assimilation. This model proved to reduce misclassifications while predicting the correct segmentation label. Notably, our model outperformed others when trained on the Cityscape dataset without soliciting any external dataset. This is a significant step in reducing the data dependency for the segmentation task. Widely used benchmark finely annotated benchmark Cityscape datasets and extensive experimental findings on the validation dataset have shown the superiority of the suggested strategy over the most advanced techniques.
